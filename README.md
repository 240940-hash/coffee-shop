# ☕ Coffee Shop Website

A modern and responsive Coffee Shop website built using HTML, CSS, and JavaScript. The website provides an attractive user interface for showcasing coffee products, menu, services, customer reviews, and contact information.

## 🚀 Features

- 🏠 Responsive Home Page
- ☕ Coffee Menu Section
- 📖 About Us Section
- ⭐ Customer Reviews
- 🖼️ Image Gallery
- 📞 Contact Form
- 📱 Mobile-Friendly Design
- 🎨 Modern UI/UX

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/240940-hash/Coffee-Shop.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
